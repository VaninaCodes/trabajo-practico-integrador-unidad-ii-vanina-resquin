## Descripción del proyecto

Sitio web institucional del **Instituto Politécnico Formosa (IPF) — Dr. Alberto Marcelo Zorrilla**, desarrollado como trabajo práctico integrador de la Unidad II.

El sitio cuenta con 4 páginas HTML interconectadas que presentan información institucional, la oferta académica, datos de contacto y una página de inicio, todo desarrollado con HTML semántico, CSS personalizado y Bootstrap 5.

## Páginas

| Archivo              | Descripción                                                                              |
| -------------------- | ---------------------------------------------------------------------------------------- |
| `index.html`         | Página de inicio con bienvenida, descripción institucional, carrusel y cards de carreras |
| `institucional.html` | Reseña histórica, misión, visión, autoridades y estructura organizativa                  |
| `carreras.html`      | Las 4 tecnicaturas con descripción, duración y requisitos de inscripción                 |
| `contacto.html`      | Formulario de consulta, datos de contacto, redes sociales y mapa de ubicación            |

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 personalizado
- Bootstrap 5.3.8 (via CDN)
- Google Maps Embed API

## Componentes de Bootstrap utilizados

- **Navbar** con toggler responsive (todas las páginas)
- **Carousel** de imágenes automático (`index.html`)
- **Cards** para presentación de carreras (`index.html`)
- **Grid system** (columnas y filas) (`contacto.html`, `index.html`)
- **Form controls** para el formulario de contacto (`contacto.html`)
- **Table** para la tabla de autoridades (`institucional.html`)
